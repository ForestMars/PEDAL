# Deploying

PEDAL supports automated deployment to a variety of environments.

## Deployment Strategies
- Blue-green deployments
- Canary releases
- Rolling updates

## Deployment Targets
- AWS (ECS, Lambda)
- Azure (App Services, AKS)
- GCP (Cloud Run, GKE)
- Kubernetes clusters

## Rollback
- Automated rollback on failure
- Workflow-based deployment recovery
