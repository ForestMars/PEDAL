# Security

PEDAL includes features and best practices to keep your workflows and data secure.

## API Security
- Rate limiting and input validation
- CORS configuration

## Encryption
- Data at rest: encrypt database and config files
- Data in transit: use TLS/SSL for all endpoints

## Audit Logs
- Track user actions and workflow changes
- Store logs securely for compliance

## Vulnerability Scanning
- Integrate with Dependabot, Snyk, or similar tools
- Regularly update dependencies
